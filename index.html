What are History Snapshots?

Every time you click Save Snapshot, the dashboard records the current rates, indexes, MBS prices, and notes. This becomes your private history. Use the dropdown above the main chart to view past trends for any individual item (30yr, 15yr, 10yr, SOFR, Spread, MBS, etc.).

This is different from the long multi-year public chart on Mortgage News Daily. Those come from public surveys. Yours is built only from the values you personally save each day, so it stays private and exact.

Tip: Update the Index & MBS fields (or click Refresh Live Indexes) from MND/FRED each morning and hit Save Snapshot. After a couple of weeks the individual charts become very useful.
